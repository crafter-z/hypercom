# HyperCom
generated by AI

一款现代化的串口调试工具，基于 Tauri v2 + React + Rust 构建。

## 功能特性

- 🔌 **串口管理** - 自动搜索、连接、配置串口
- 📊 **数据收发** - 支持 Hex/ASCII 格式显示和发送
- ⚡ **高性能** - Rust 后端处理，数据节流避免 UI 卡顿
- 🎨 **现代化 UI** - React + Tailwind CSS，支持亮色/暗色主题
- 💾 **命令组** - 自定义命令组管理（开发中）
- 📝 **日志存储** - 日志记录与导出（开发中）

## 技术栈

### 前端
- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Zustand** - 状态管理
- **@tanstack/react-virtual** - 虚拟滚动
- **Lucide React** - 图标库

### 后端
- **Tauri v2** - 桌面应用框架
- **Rust** - 系统编程语言
- **serialport-rs** - 串口通信库
- **tokio** - 异步运行时
- **serde** - 序列化框架

## 快速开始

### 环境要求

- **Node.js** >= 18
- **Rust** (通过 [rustup](https://rustup.rs/) 安装)
- **Visual Studio Build Tools** (Windows，需要"使用 C++ 的桌面开发"工作负载)

### 安装依赖

```bash
# 安装前端依赖
npm install
```

### 开发模式

```bash
npm run tauri dev
```

此命令会：
1. 启动 Vite 开发服务器（前端热更新）
2. 编译 Rust 后端
3. 打开应用窗口

### 生产构建

```bash
npm run tauri build
```

构建完成后，安装包位于：
```
src-tauri/target/release/bundle/
├── msi/          # Windows 安装包 (.msi)
└── nsis/         # NSIS 安装包 (.exe)
```

## 使用说明

### 串口连接

1. 点击串口下拉框选择要连接的串口
2. 配置串口参数（波特率、数据位、停止位、校验位）
3. 点击"连接"按钮建立连接
4. 连接成功后状态指示灯变绿

### 数据发送

1. 选择发送格式（HEX 或 ASCII）
2. 在输入框中输入数据
3. 按 Enter 或点击"发送"按钮发送数据

### 数据接收

- 接收到的数据会自动显示在数据区域
- 支持虚拟滚动，可处理大量数据
- 显示时间戳和数据方向（RX/TX）

## API 命令

| 命令名 | 说明 |
|--------|------|
| `list_ports` | 列出可用串口 |
| `open_port` | 打开串口 |
| `close_port` | 关闭串口 |
| `send_data` | 发送数据 |
| `get_status` | 获取串口状态 |
| `get_settings` | 获取应用设置 |
| `save_settings` | 保存应用设置 |
| `list_command_groups` | 获取命令组列表 |
| `save_command_group` | 保存命令组 |
| `delete_command_group` | 删除命令组 |
